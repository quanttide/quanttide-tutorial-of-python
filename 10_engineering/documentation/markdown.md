# Markdown简介

## Markdown是一门标记语言

标记语言。
Markdown的主要好处是不需要手动排版，这样可以更专心写内容。

## 标题

```markdown
# 一级标题
## 二级标题
```

Markdown支持1-6级标题。

## 无序列表

```markdown
- 我是小明
- 我是小红
```

## 有序列表

```markdown
1. 我是第一条
2. 我是第二条
```

## 超链接

```markdown
[量潮课堂](https://class.quanttide.com)
```

## 图片

```markdown
![图片无法加载时显示我](example.png)
```

和超链接的主要区别是前面多一个英文感叹号`!`。
